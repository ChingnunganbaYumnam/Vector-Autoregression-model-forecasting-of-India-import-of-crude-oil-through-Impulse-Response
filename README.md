# Vector-Autoregression-model-forecasting-of-India-import-of-crude-oil-through-Impulse-Response
This the stata code for VAR showing how India import of Crude oil response to shocks in key variables namely real price of imports, real gdp, alternative energy uses
The datasets is the gives variables values that I have use collected from Indiastat and RBI
The text file is the code for the VAR model
